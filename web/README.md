# MassiMappi website

## struktuuri

Webapissa kaikki muokattava löytyy kansion `src` alta. alla olevat titlet ovat relatiivisia tähän kansioon.

Firebase-setup on identtinen puhelinapplikaation kanssa. By default, tämä olettaa, että tiedosto `firebaseConfig` löytyy tämän repositorion rootista, mutta muokkaamalla tiedostoa `src/firebase-auth.js` voi sitä vaihtaa. Verkkosivuversio ja puhelinversio voi siis toimia toisistaan erillisinä.
